# Logo Assets for warp-md Documentation

## Files Created:
- `logo.svg` - Main logo (200x60px) with wave design and brand colors
- `favicon.svg` - Small favicon version (32x32px) for browser tabs

## Color Scheme:

### Light Mode:
- **Primary**: Resolution Blue `hsl(216, 100%, 25%)`
- **Background**: White Ice `hsl(204, 71%, 96%)`
- **Text**: Ebony Clay `hsl(238, 27%, 21%)`

### Dark Mode:
- **Primary**: Light Resolution Blue `hsl(216, 100%, 50%)`
- **Background**: Deep Ebony Clay `hsl(238, 27%, 10%)`
- **Text**: White Ice `hsl(204, 71%, 96%)`

## GitBook Setup Instructions:

1. **Logo**: Go to GitBook → Site Settings → Branding → Logo → Upload `logo.svg`
2. **Favicon**: Go to GitBook → Site Settings → Branding → Favicon → Upload `favicon.svg`
3. **Colors**: Go to GitBook → Site Settings → Appearance → Theme
   - Set Primary Color to: `hsl(216, 100%, 25%)`
   - Set Background Color to: `hsl(204, 71%, 96%)` (light) / `hsl(238, 27%, 10%)` (dark)

## Logo Design Notes:
- Wave pattern represents molecular dynamics "warping"
- Scalable SVG format for crisp display at any size
- Uses brand colors for consistency
- Simple, professional design suitable for technical documentation
